This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

- Run Locally
  ```
  npm run dev
  ```
- Build for specific path, setting `<base href=my-path>`, and outputting to ./out/my-path
  ```
  export PAGES_BASE_PATH=my-path; npm run build;
  ```
- Test serving from paths
